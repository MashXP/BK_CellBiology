# Cell Biology - Study Materials

This repository contains study materials for "Essential Cell Biology, 5th Edition". It includes notes, summaries, questions, and a quiz application to aid in learning the concepts presented in the textbook.

## Book Citation

Alberts, B., Bray, D., Hopkin, K., Johnson, A., Lewis, J., Raff, M., Roberts, K., & Walter, P. (2019). *Essential Cell Biology* (5th ed.). W. W. Norton & Company.

## Directory Structure

*   ~~`pdf/ , /chapters`: Contains PDF presentations for each topic.~~ Will be hosted privately due to licensing issues.
*   `_questions/`: Stores markdown files for questions related to each chapter.
*   `_summary/`: Stores markdown files for summaries of each chapter.
*   `quiz/`: Contains files for a my web-based quiz application: https://mashxp.github.io/quiz_webapp/
*   `bookmarks.json`: Stores bookmarks for the textbook.
*   `split_chapters.py`: A script to split the main PDF into individual chapter PDFs.
